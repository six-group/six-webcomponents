Checkboxes allow the user to toggle an option on or off.

Checkbox

This component doesn't work with standard forms. Use six-form instead.

Please refer to the section on form control validation to learn how to do client-side validation.

Examples

Checked

Use the checked attribute to activate the checkbox.

Indeterminate

Indeterminate

Disabled

Labels

Use the label attribute to give the checkbox an accessible label. For labels that contain HTML, use the labelslot instead.
