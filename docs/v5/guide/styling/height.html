Height ​

The Six Webcomponents library includes height tokens for consistent component sizing.

CSS VariableValuePreview
--six-height-small1.875rem (30px)
--six-height-medium2.5rem (40px)
--six-height-large3.125rem (50px)

Example Usage ​

css
.button-small {
  height: var(--six-height-small);
}

.input-medium {
  height: var(--six-height-medium);
}
